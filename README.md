## Getting Started

1. **Install dependencies:**

   ```bash
   npm install
   ```

2. **Add your images:**

   - Add your profile photo as `public/profile.png`
   - Add project screenshots to `public/projects/`:
     - `rent-1.png` - Main rental app screenshot
     - `rent-2.png` - Secondary screenshot
     - `rent-3.png` - Feature screenshot
     - `rent-4.png` - Additional screenshot

3. **Run the development server:**

   ```bash
   npm run dev
   ```

4. **Open [http://localhost:3000](http://localhost:3000)**

## Customization

- **Colors:** Edit `tailwind.config.ts` to customize the color palette
- **Content:** Edit component files in `src/components/` to update text and links
- **Social Links:** Update GitHub, LinkedIn, and email links in `Hero.tsx` and `Contact.tsx`

## Build for Production

```bash
npm run build
npm start
```
